Louis's Cellular Automaton Generator

To use this program,
The 1st option is a positive integer between 0 and 255 that specifies the rule number
The 2nd option is a positive integer that specifies the amount of rows you want
The 3rd option is a positive integer that specifies the amount of column's you want
The 4th option specifies the styling of the first row
The 5th option only applies to odd column values and specifies the starting digit as either left or right
